# Lolisoft OrderFlow Import and Sync for Allegro and WooCommerce

**Turn WooCommerce into your Allegro operations hub.** Import offers, keep stock and prices aligned, pull Allegro orders into WooCommerce, and manage day-to-day sync work from one admin panel.

> **Version:** 1.3.7 &nbsp;|&nbsp; **WordPress:** 6.0+ &nbsp;|&nbsp; **WooCommerce:** 8.0+ &nbsp;|&nbsp; **PHP:** 7.4+

## Value Proposition

Lolisoft OrderFlow Import and Sync for Allegro and WooCommerce helps merchants remove manual marketplace work. Instead of updating Allegro and WooCommerce separately, the plugin centralizes catalog sync, stock sync, price sync, and order import in one workflow.

## Target Customer

This plugin is built for:

- WooCommerce stores already selling on Allegro
- Merchants who need fewer stock mismatches and fewer manual updates
- Small and mid-sized teams that want one operational dashboard
- Agencies and operators managing multiple Allegro seller accounts

## What the Product Includes

- OAuth 2.0 connection to Allegro
- Product import from Allegro to WooCommerce
- Product content sync for linked offers (titles, descriptions, and images) in configurable directions
- Stock and price sync in configurable directions
- Order import into WooCommerce
- Multi-account support
- Analytics, inventory planning, health checks, logs, and sync history
- Background processing through WP-Cron
- Compatibility with HPOS and Cart/Checkout Blocks

## Quick Installation

1. In WordPress go to **Plugins → Add New → Upload Plugin**.
2. Upload the packaged ZIP that contains `allegro-sync.php`.
3. Activate the plugin.
4. Open **Allegro Sync → Settings**.
5. Enter your Allegro Client ID and Client Secret.
6. Connect your Allegro account and run the first sync from the dashboard.
